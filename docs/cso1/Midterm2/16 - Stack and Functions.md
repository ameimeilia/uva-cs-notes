# Stack and Functions
## Writing Assembly for x86 Processors
- `RAX`: 64 bits
- `EAX`: lower 32 bits
- `AX`: lower 16 bits
- `AH`: higher 8 bits of the lower 16 bits
- `AL`: lower 8 bits of the lower 8 bits
